## prerequisites

pip 21.2.4

python 3.9

## Installation

```bash
pip install SpeechRecognition==3.8.1
pip install pydub==0.25.1
pip install librosa
pip install sklearn
pip install flask
pip install werkzeug=2.0.1
pip install pyaudio=0.2.11
```
create a folder "audio-chunks" inside uploads folder
